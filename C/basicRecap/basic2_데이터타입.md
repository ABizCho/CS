## C 변수의 자료형 Data Type
C언어에서 변수는 크게 `정수형 변수`, `실수형 변수` 두가지로 분류된다.


- 정수형 변수: 정수의 저장을 목적으로 선언된 변수
- 실수형 변수: 소수점 이하의 값을 지니는 실수의 저장을 목적으로 선언된 변수

```c
int num1 = 24;

double num2 = 3.14;
```
<br>

정수냐, 실수냐에 따라 값이 메모리 공간에 저장 및 참조되는 방식이 다르기 때문이다.

크기에 따라, 이들은 아래처럼 분류된다.

- 정수형 변수
  - char형
  - short형
  - int형
  - long형

- 실수형 변수
  - float형
  - double형
